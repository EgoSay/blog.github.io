搭建博客记录

Author: EgoSay Date: October 9, 2019  15:48:49 Category: Life Words: 600

前言

  • 博客基于MIT License开源于GitHub, 感谢
  • 博客引擎来自于hexo, 感谢
  • 博客模版来自于Siricee的Chic, 感谢
  • 动态背景来自于canvas-colorful-nest.js, 感谢
  • 书单页面来自于插件hexo-douban, 感谢
  • CDN加速服务来自于Cloudflare, 感谢

Why?

为什么选择搭建这样一个博客呢？

  1. 以前在CSDN、简书等平台上也会写博客去记录自己的一些学习总结, 但难免自己的一些小心理满足不了, 还是自己的博客自己随便折腾的好哈哈
  2. 也会记录一些自己的经历成长吧, 自己有时候难免胡思乱想吧
  3. 自己也挺喜欢看书的, 所以整合了豆瓣书单的插件做了一个书单页面, 工作后没有太多时间了, 但是我发现每天看几页, 两个多月吧已经看完了好几本书了, 觉得后面有时间再总结一下写写读书笔记
  4. 经常会有一些想法, 过了时间也容易忘, 以后就在这里写下来吧, 对啦, 以前写的博客有时间的话也会慢慢迁移到这里, 慢慢来吧

What?

做什么样的博客？

  1. 简简单单就好, 包括模版主题也是简约风格, 没有什么酷炫的动画啊, 一些我觉得不是必要的功能也都没开, 让自己专注踏实一点吧, 有时间就写点东西挺好的
  2. 在这里可能会看到一些我的胡思乱想, 所以评论留言啊, 赞赏什么的功能也都没有, 当然欢迎大家来认识交个朋友, GitHub留言或者邮箱都行, 后面看看再留些其他联系方式吧

How?

怎么搭建的这个博客？

  1. 从开始想搭建博客的想法到诞生差不多一周的时间吧, 想过一些方案,比如自己买服务器搭独立站点, Jekyll, Hexo, Hugo等等,最后选择了 hexo + github pages
  2. 博客全部都是基于开源方案实现, 基本上就是前言提到的那些, 再次感谢伟大的开源贡献者们
  3. 后面可能还会添加一些小功能完善改进一下, 比如统计分析的插件啊, 页面的一些优化啊等等, 反正生命在于折腾哈哈
