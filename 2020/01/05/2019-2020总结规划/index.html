2019~2020总结规划

Author: EgoSay Date: January 5, 2020  17:53:15 Category: Life Words: 2.1k

2019

2019大学毕业，算是真正踏入社会职场的第一年。

  • 19年有后悔的事吗？
    当然，后悔年初的时候没有把握机会春招进入大厂，后悔没有来一场毕业旅行，后悔没有听晗哥大佬的话及时换工作…..还有许多小事情现在看来也是有遗憾的
  • 19年觉得自己有哪些变化呢？
    一个是心态吧，刚毕业抱着满怀热情，虽说没有被社会打脸教训,但却也是慢慢认识到自己还是太浮躁了。二是沟通力吧，出差的时候一个人在客户公司那边呆一个多礼拜，迫不得已跟客户沟通，在公司负责环境上线跟同时沟通，有时候真的想骂人，但还是忍住了，抱怨不如适应，遗憾的是也只是适应了但却没有寻求改变。三是思维方式的转变吧，工作上认识到要看全局抓细节，自己的事及时完成做好的同时要及时汇报；人际关系方面，知道了要换位思考；遇到问题的时候不要慌张，要沉着思考，和别人协作时人家需要的只是解决方案，不是你的解释推脱
  • 19年有什么欢喜的事么？
    毕业啦，自己终于开始赚钱啦。实习的小伙伴还在一块儿然后转正一起工作，没事还能一起吃吃火锅，北京有了高中同学，可以一起聊天。

读书

看完了：

  • 《富爸爸，穷爸爸》
  • 《人性的弱点》
  • 《阿里传》
  • 《自卑与超越》
  • 《未来简史》
  • 《一本书看透股权架构》
  • 《金字塔原理》
  • 《从零开始读懂金融学》
  • 《原则》

技术

大数据技术

  • Spark进阶：主要还是集中在PySpark，另外看了一点调度系统的源码
  • Kafka: 算是学的比较深入系统了，可还是感觉缺很多，多实践吧
  • Flink：刚刚开始学….

编程语言

  • Python精通：学了一点爬虫，学了一点数据分析，跟着《流畅的Python》深入学习了一部分内容
  • Java深入：多线程，虚拟机，源码还差了许多
  • Scala/Go：Scala函数式编程确实比较难，需要花时间啃

其他

  • 容器、K8s：停留在会一点docker命令的基础上，最主要的调度，网络配置没有系统去实践过，K8s看了一点点九牛一毛
  • GitHub参与：坚持每天最少一个commit，虽然大部分时候都很水，不过把这个习惯养成坚持下来了还可以
  • 写作（博客、公众号）：博客搭建完了，写了几篇博客，公众号却还是没开始行动

综合素质

  • 分享、演讲、沟通能力：虽然知道这是自己的一大缺点，但却基本没怎么作出行动去改变
  • 业务分析、场景分析能力、产品视角：找不到突破点，甚至连什么是产品思维还没搞懂
  • 理财知识学习：今年理财没赚多少钱，也没有系统去学习了解
  • 自律习惯培养：英语学习坚持了几个月还是半途而废了，还有一些小想法也大部分都是激情一过就不了了之了，唯一还算可以的应该就是GitHub坚持下来了
  • 独立思考、总结分析能力：以为不刷微博抖音热点，选择订阅优质公众号，付费加入一些知识星球就可以，但发现这种最多算是提高了你的信息源门槛，也不能完全相信，就像看书一样，尽信书不如不读书
  • 情绪控制、少冲动、更理性：做了一些冲动的选择，到现在为止还觉得有点后悔。

自我认知

  • 先去真正认识自己，做了盖洛普优势识别器测试、16种人格测试，也算对自己有了一个初步的了解了吧
  • 认识到有两个我，《原则》里是这样说的：”较低层次的我就像经过训练的斗犬，始终保持战斗意识，而较高层次的自我想把事情搞明白”，”有逻辑和有意识的自己”与”情绪化和潜意识的自己”一直在争夺大脑的控制权，要学会去理解自己的思维盲点障碍

人生规划

  • 想过考研，但却下不了决心直接辞职去考研，工作三年与读研三年，我也不知道于我而言哪个选择对以后的人生影响更大
  • 关于工作，每次有想换工作的想法的时候，最后却又会发现现在工作其实还不错，也不清楚是自己被麻痹了还是确实是太浮躁了，过完年回来试一下吧，看看自己这一年里到底成长了多少吧
  • 城市选择，看到一系列文章说深圳的发展前景，心动了想回深圳
  • 副业开展，加入了一些社群，看到许多其他人发展副业，做斜杠青年，心动但却没有付出任何行动，无怪乎只能凭白羡慕。

2020规划

2020年，新一个10年的开启，想看看自己10年后是什么样啊。

读书

继续坚持每天看书, 每个月一本，一年最少看完12本书

技术

大数据技术

  • 流计算Flink/StructStreaming的学习
  • 项目实践：（1）离线/实时计算（2）推荐系统
  • 源码学习

编程语言

Java继续深入，Scala函数式编程花时间去啃

其他

  • 容器、K8s：真正的去实践容器调度编排，会写K8s的yml文件，学会利用K8s搭建起一个大数据环境供学习
  • GitHub参与：继续坚持每天最少一个commit，年度突破1000个 commit
  • 写作（博客、公众号）：每月最少两篇博客，预计5月份吧开始想想怎么运营公众号，先找好定位，争取每月写上一篇
  • 算法和数据结构，坚持每周刷题
  • 计算机基础，操作系统/网络方面的体系学习

综合素质

  • 分享、演讲、沟通能力：先从家里人沟通开始，最少每周给家里打个电话。然后尝试走出自己的舒适圈，多去认识一些其他朋友，多参加户外活动
  • 业务分析、场景分析能力、产品视角：先从商业视角去了解，比如这个产品为什么火，它的运营策略如何，做一个产品先从想好怎么赚钱开始
  • 理财知识学习：尝试港股打新，践行 E大的网格策略，学会看财报，坚持自己的生活记账，再就是看一些商业书籍吧
  • 自律习惯培养：坚持早起看书，坚持GitHub提交，坚持每晚回家学习，坚持俯卧撑运动
  • 独立思考、总结分析能力：多总结，多反思，每月、每周给自己留时间复盘一下，另外就是看完书后尝试去写读后感
  • 情绪控制、少冲动、更理性：借鉴一下《原则》里提到的，每次下决定之前先想想基于什么逻辑下的这个决定，记下来，最后对比结果，错了、后悔了反思问题在哪总结成日志

自我认知

  • 做了盖洛普优势识别器测试，我的五大主题：思维、责任、关联、搜集、排难，结合16种人格测试结果INFJ型人格，目前来看自己的优势在于思考，有责任感，靠谱令人信服，希望能结合这些优势快速成长
  • 理解自己的思维盲点障碍，让两个我之中的更高层次的我占据主导，自控力up,up up!
  • 心态上的转变，适应职场，杜绝玻璃心，踏踏实实做。

人生规划

  • 年后回来看机会，看能不能进大厂，不然的话准备考研，目标选择北京/上海/深圳的高校
  • 副业开展，精力是有限的，今年专注于自己的技术/个人综合素质的提升上，副业的话先从理财开始，可以尝试一下股票量化分析方面的探索
  • 给家里配置好保险，如果工作顺利能进入理想公司的话，下半年可以考虑找个女朋友，谈场恋爱, 嘻嘻
